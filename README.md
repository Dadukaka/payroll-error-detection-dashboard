# Payroll Error Detection Dashboard

A Streamlit dashboard to detect payroll errors efficiently.

## Features
- Detects missing or incorrect payroll entries
- Highlights errors for review
- Interactive dashboard built with Streamlit

## Files
- `app.py` → main Streamlit app
- `payroll-csv` → example input CSV
- `payroll-error` → example error file
